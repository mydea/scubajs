    ajax

    ()

    Defined in src\Scuba.js:524

    This function is used instead of the regular $.ajax() function It will route the requests to the local db and perform any actions required there It will also queue changing requests (post, put, delete) and send them to the server this happens asychronously - results are returned from the local db immediatley, and changes are performed on the server at some point later The original ajax function (for external requests) is available via ajaxOnline

    Returns:

    promise A promise which can be resolved (which will contain the returned data)
